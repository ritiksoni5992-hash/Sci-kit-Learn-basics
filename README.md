# Scikit-Learn Basics

A beginner-friendly Python project that demonstrates commonly used data analysis and machine learning functions from Pandas and Scikit-Learn. This project is designed to help newcomers understand the basic workflow of data exploration, preprocessing, model training, prediction, and evaluation.

## Features

### Data Exploration

* `head()`
* `describe()`
* `info()`
* `shape`
* `columns`
* `value_counts()`
* `isnull()`

### Machine Learning Basics

* `train_test_split()`
* `fit()`
* `predict()`
* `score()`
* `accuracy_score()`

### Example Model

* Decision Tree Classifier
* Train/Test Data Splitting
* Model Evaluation

## Technologies Used

* Python
* Pandas
* Scikit-Learn

## Installation

```bash
git clone https://github.com/yourusername/sklearn-basics.git

cd sklearn-basics

pip install pandas scikit-learn
```
